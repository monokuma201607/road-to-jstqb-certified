# Base Profile: JSTQB Advanced Level TA & AI-Driven Process Manager Candidate

## 1. 最終ゴール (Final Goal)
- **目標試験**: JSTQB Advanced Level Test Analyst (AL TA) 合格
- **目標レベル**: 高度なテスト設計のスペシャリスト・次世代QAリーダー
    - 複雑な仕様に対し、直交表、ペアワイズ、ドメイン分析等の高度なブラックボックス技法をK4レベルで適用できる。
    - ユーザー要件やユースケースから、ビジネスリスクを最小化するテストを設計・分析できる。
    - 自身の高度な設計能力をベースに、AIを活用したテスト自動生成やプロセス改良を主導できる（TMへの布石）。

## 2. 学習スタイル (Learning Style)
- **メイン教材**: 
    - JSTQB Advanced Level Test Analyst シラバス
    - AL TA 向け長文シナリオ問題集 / ISTQB Glossary
- **トレーニング手法**:
    - **Scenario-based Analysis**: 実務に近い複雑な要件（制約、競合状態など）を含む長文シナリオを読み解き、最適なテスト技法を選択・適用する。
    - **AI-Collaborative Design**: AIを壁打ち相手とし、「どうすればこの組み合わせテストを極限まで圧縮できるか」「AIプロンプト化するならどう要件を定義するか」を議論する。
    - **Peer Review Simulation**: 欠陥報告や要件レビューにおけるTAとしての役割（ユーザー視点・ドメイン視点）をシミュレーションする。
- **管理手法**: Git による学習ログのバージョン管理と、AIプロンプト/テストスクリプトの資産化。
