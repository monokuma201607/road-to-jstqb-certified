# Base Profile: JSTQB Foundation Level Specialist

## 1. 最終ゴール (Final Goal)
- **目標試験**: JSTQB Certified Tester Foundation Level (Version 4.0) 合格
- **目標レベル**: プロフェッショナル・テスト技術者
    - シラバスの用語（エラー・欠陥・故障など）を厳格に使い分けられる。
    - 7つのテスト原則を実務のコンテキストに当てはめて説明できる。
    - テスト技法（ブラックボックス/ホワイトボックス）を用いて、適切なテストケースを設計できる（K3レベル）。
    - 開発ライフサイクル（アジャイル/シーケンシャル）に応じたテストアプローチを提案できる。

## 2. 学習スタイル (Learning Style)
- **メイン教材**: 
    - JSTQB Foundation Level シラバス Version 4.0.J02
    - ISTQB/JSTQB 用語集 (Glossary)
    - JSTQB公式 模擬試験問題
- **トレーニング手法**:
    - **Logical Discussion**: シラバスの概念（例：シフトレフト、独立性）について、AIエージェントと実務シナリオベースで議論する。
    - **Test Design Exercise**: 境界値分析、同値分割法、決定テーブル等の技法を用いた計算・設計演習。
    - **Terminology Drill**: 紛らわしい用語（確認テスト vs リグレッションテスト等）の定義を言語化する。
- **管理手法**: Git による学習ログのバージョン管理（`01_Progress_Log/Daily_Lessons/`）。
